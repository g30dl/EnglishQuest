EnglishQuest
============

EnglishQuest es una app educativa con enfoque de gamificación para practicar vocabulario, gramática y listening en inglés. Los usuarios avanzan por niveles, desbloquean lecciones y ganan XP al completar preguntas.

Qué ofrece
----------
- Lecciones agrupadas por nivel con estados desbloqueado/bloqueado y completado.
- Progreso visible y recompensas de XP para motivar el avance.
- Modalidades variadas de preguntas: opción múltiple, escritura y listening con feedback inmediato.
- Experiencia consistente con tarjetas, badges y estados vacíos reutilizables.

Cómo empezar
------------
1. Instala dependencias (`npm install`).
2. Inicia el servidor de desarrollo (`npm run start`).
3. Abre la app en dispositivo/emulador con Expo Go para probar las lecciones.

Estructura rápida
-----------------
- app/(drawer)/(tabs)/: pantallas principales de vocabulario, gramática y listening.
- components/: tarjetas y elementos UI reutilizables para mantener coherencia visual.
- context/ProgressContext: datos de áreas, niveles, lecciones y progreso del usuario.

Iconos y estilo
---------------
La app usa íconos Ionicons y una paleta verde centrada en claridad y accesibilidad.
